```cpp
void InsertSort(Sqlist &L)
{
    int i;
    int j;
    for (i = 2; i <= L.length; i++){
        if (L.r[i] < L.r[i-1])
        {
            L.r[0] = L.r[i];
            for (j = i-1; L.r[0] < L.r[j]; j--)
            {
                L.r[j+1] = L.r[j];
            }
            L.r[j+1] = L.r[0];
        }
    }
}
```

### 堆排序

什么是堆：  
若n个元素的序列a_1, ..., a_n满足：  
a_i < a_2i;  
a_i < a_{2i+1};  
则称该序列为小根堆；  
a_i > a_2i;  
a_i > a_{2i+1};  
则称该序列为大根堆；

***
>从上述定义可以看出，堆实质上是一棵完全二叉树，并且满足二叉树中任一非叶子结点均小于（大于）它的孩子结点。
